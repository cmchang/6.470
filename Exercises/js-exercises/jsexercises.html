JavaScript Exercises 0 - 4

Exercise 0: Importing JavaScript files

The file containing the jQuery code has already been imported for you. Add a line to import the file "jsexercises.js" too.

Exercise 1: Debugging JavaScript

There is a syntax error in the JavaScript file which you'll need to fix before you can see exercises 2 and 3. Open the debugger in your browser to try and find the error. (If you don't know how to open the debugger in your browser, ask someone or consult your favorite search engine.) Fix it, refresh, and the next two exercises should appear.

Exercise 2: I see dead people glowing squares!

There is a CSS rule targeting the "glow" class which makes the background color of an element green. Without changing the CSS at all, accomplish the following:

  • Make the three tiles below glow green when your mouse hovers over them.
  • Make the tiles return to their original color when the mouse is no longer on top of the tile

You may find the jQuery effects list useful.

Exercise 3: Start talking to yourself

We don't know how to make a chat room for multiple people yet, but we sure can talk to ourselves!

When the user clicks the submit button, you will need to read the text in the name and message fields, and create a new element to append to the list of messages.

You may find the jQuery DOM insertion and DOM manipulation lists useful.

Bonus: Automatically scroll the chat window when it fills, rather than have the list keep growing. You may have to change the CSS files too for this one.
